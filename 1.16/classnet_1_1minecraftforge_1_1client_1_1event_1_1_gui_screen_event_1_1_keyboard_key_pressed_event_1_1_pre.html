Detailed Description

Fired before the key press is handled by the screen.

This event is cancelable and does not have a result. If the event is cancelled, the screen's key press handler will be bypassed and the corresponding KeyboardKeyPressedEvent.Post will not be fired.

This event is fired on the main Forge event bus, only on the logical client.
