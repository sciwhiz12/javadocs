Detailed Description

Fired after world rendering is finished.

This can be used for custom rendering outside of e.g. a tile entity or entity renderer.

Rendering in this event occurs after all world rendering (after the render buffers are flushed); interactions with translucent rendering and other special effects may have unavoidable side-effects.

An instance of IRenderTypeBuffer can be retrieved from RenderTypeBuffers::getBufferSource(). The caller is responsible for flushing the buffers after use, through RenderTypeBuffers.Impl::finish(). Not flushing the buffers will cause rendering issues.

This event is not cancellable, and does not have a result.

This event is fired on the main Forge event bus, only on the logical client.
