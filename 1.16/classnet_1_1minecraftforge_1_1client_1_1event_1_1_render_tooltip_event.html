◆ getStack()

public ItemStack RenderTooltipEvent.getStack ( )

Returns the ItemStack for which the tooltip is being rendered.

In certain situations where a tooltip is rendered without an ItemStack, this will be the value of GuiUtils::cachedTooltipStack, which has the value of ItemStack::EMPTY.

Returns
The stack which the tooltip is being rendered for, may be ItemStack::EMPTY

◆ getLines()

public List<? extends ITextProperties > RenderTooltipEvent.getLines ( )

Returns an unmodifiable view of the lines of text on the tooltip, as represented by ITextProperties.

This may change between RenderTooltipEvent.Pre and RenderTooltipEvent.Post.

Use ItemTooltipEvent to modify the text of the tooltip.

Returns
An unmodifiable view of list of text on the tooltip
