Detailed Description

Fired for registering item colors handlers.

The block colors should only be used for referencing or delegating item colors to their respective block colors. Use ColorHandlerEvent.Block for registering your block color handlers.

This event is not cancellable, and does not have a result.

This event is fired on the mod-specific event bus, only on the logical client.

See also
ForgeHooksClient::onItemColorsInit(ItemColors, BlockColors)
