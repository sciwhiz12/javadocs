Detailed Description

Fired when a mouse scroll wheel is used outside of a screen and a player is loaded, before being processed by vanilla code.

This event is cancellable, and does not have a result. If the event is cancelled, then the mouse scroll event will not be processed further.

This event is fired on the main Forge event bus, only on the logical client.

See also
ForgeHooksClient::onMouseScroll(MouseHelper, double)
the online GLFW documentation
