Detailed Description

Fired when the client player logs in to the server.

When this event is fired, the player should be initialized at that point in time.

ClientPlayerNetworkEvent.getController(), ClientPlayerNetworkEvent.getPlayer(), and ClientPlayerNetworkEvent.getNetworkManager() are guaranteed to never return null in this event.

This event is not cancellable, and does not have a result.

This event is fired on the main Forge event bus, only on the logical client.
