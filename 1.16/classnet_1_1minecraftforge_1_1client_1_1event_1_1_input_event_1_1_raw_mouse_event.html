Detailed Description

Fired when a mouse button is clicked, before being processed by vanilla code.

This event is cancellable, and does not have a result. If the event is cancelled, then the mouse event will not be processed by vanilla code (e.g. keybinds and screens).

This event is fired on the main Forge event bus, only on the logical client.
