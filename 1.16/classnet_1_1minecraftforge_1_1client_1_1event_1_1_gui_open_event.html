Detailed Description

Fired before any Screen will be opened, for changing it or preventing it from being opened.

This event is cancelable, and does not have a result. If this event is cancelled, then the Screen shall be prevented from opening and the previous screen shall remain.

This event is fired on the main Forge event bus, only on the logical client.

See also
Minecraft::displayGuiScreen(Screen)
