Detailed Description

Fired when the ModelManager is notified of the resource manager reloading.

Called after model registry is setup, but before it's passed to BlockModelShapes for block model baking. This can be used to modify and wrap models before they are sent for model baking.

This event is not cancellable, and does not have a result.

These events are fired on the mod-specific event bus, only on the logical client.

See also
ForgeHooksClient::onModelBake(ModelManager, Map, ModelLoader)
