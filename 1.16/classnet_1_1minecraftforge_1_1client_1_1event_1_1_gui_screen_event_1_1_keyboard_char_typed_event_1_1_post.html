Detailed Description

Fired after the character input is handled, if not handled by the screen and the corresponding KeyboardCharTypedEvent.Pre is not cancelled.

This event is cancelable, and does not have a result. If the event is cancelled, the character input will be set as handled.

This event is fired on the main Forge event bus, only on the logical client.
